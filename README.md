# Memory-Card-Puzzle
> Language: Racket

Take turns along with an AI player to pick up two cards and see if you have a 
matching pair, the first person who gets the last correct pair win the game, 
meanwhile for the other players a losing screen appears for them.

________________________________________________________________________________
Instructions for players:

1-5 -> Used to pick cards from the top row in order

6-0 -> Used to pick cards from the bottom row in order
